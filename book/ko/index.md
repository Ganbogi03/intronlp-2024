이 강좌는 자연어처리(NLP)의 기본 개념과 최신 언어모형 기술을 소개합니다. 학생들은 텍스트 처리의 기초부터 시작하여 고급 언어모형 API 활용 및 NLP 애플리케이션 개발까지 실습 중심으로 학습합니다. 특히 대규모 언어모형(LLM)의 활용과 프롬프트 엔지니어링에 중점을 두어, 최신 NLP 기술의 실무 적용 능력을 키웁니다.
